# 🎯 就活サポートサービス 要件定義書（ハッカソン向け）

## 1. 概要
学生が就職活動で必要な以下の機能を一元管理できる支援サービスです。  
AI（Gemini）を活用して添削や要約を実現し、効率的かつ深い自己理解・企業理解を進めることを目的とします。

- ES作成
- 面接準備・振り返り
- スケジュール管理

## 2. 利用ユーザー
- 主に大学生・大学院生（就活中の学生）
- 技術系・ビジネス系問わず
- 就活初期〜最終面接まで幅広く支援

## 3. 主な機能一覧
| 機能名           | 概要                                                                 |
|------------------|----------------------------------------------------------------------|
| **ログイン機能**  | Googleアカウントによるログイン。初回ログイン時にプロフィールを設定。 |
| **ダッシュボード画面** | ユーザーの活動状況を一覧表示。共通ES、登録企業、予定の一覧が確認可能。 |
| **プロフィール編集** | 名前、学校、希望職種などの編集機能（オプション）。                     |
| **ES作成・管理**   | 自己PRや志望動機などを保存・閲覧・編集。Gemini APIによるAI添削機能を搭載。 |
| **面接記録・要約** | 音声をアップロードし、AIで要約・面接官メモを付与。                     |
| **カレンダー機能** | 各企業ごとのES提出期限、面接予定などを表示。Google Calendar連携（任意）。 |
| **企業別ページ**   | 各企業ごとのES、面接、スケジュールを一覧表示。                         |

## 4. 画面一覧
| 画面名             | URLパス                  | 内容                                                                 |
|--------------------|-------------------------|----------------------------------------------------------------------|
| **ダッシュボード**   | `/dashboard`            | 簡易プロフィール、共通ES一覧、登録企業一覧、今週の予定など             |
| **プロフィール**     | `/profile`              | ユーザー情報の編集（任意）                                            |
| **ES一覧画面**      | `/es`                   | 共通/企業別ESの作成・表示・Geminiによる添削機能付き                   |
| **面接ログ画面**    | `/interviews`           | 面接の録音/要約表示、面接官の情報やメモを紐付け                      |
| **カレンダー画面**   | `/calendar`             | 選考日程（ES締切、面接、合否連絡）をカレンダー形式で表示              |
| **企業詳細画面**    | `/companies/:id`        | 選択した企業に関するES・面接・予定をまとめて表示                      |

## 5. 非機能要件
- **認証基盤**: Firebase Authentication（Google OAuth連携）
- **データ永続化**: GCP Cloud SQL（PostgreSQL）
- **AI連携**: Gemini API（ES添削、面接要約）
- **ストレージ**: Google Cloud Storage（音声ファイルなど）
- **API設計**: RESTful（Go）、フロントはNext.js
- **デプロイ先**: GCP Cloud Run or Vercel（FE）

## 6. 優先順位（ハッカソン向けMVP）
| 優先度 | 機能                                                                 |
|--------|----------------------------------------------------------------------|
| **高** | ログイン、ES作成・添削、音声要約（面接ログ）、企業登録・ダッシュボード表示 |
| **中** | カレンダー表示（モックでOK）、企業別詳細画面                           |
| **低** | Google Calendar連携、プロフィール編集                                 |

## 7. 技術スタック（予定）
| 分類           | 使用技術                          |
|----------------|----------------------------------|
| **フロントエンド** | Next.js (App Router), Tailwind CSS |
| **バックエンド**   | Go（Clean Architectureベース）     |
| **AI連携**       | Gemini API                       |
| **音声処理**     | Whisper or Gemini（音声→テキスト） |
| **ストレージ**     | GCP Cloud Storage                |
| **認証**         | Firebase Authentication          |
| **データベース**   | GCP Cloud SQL                    |
| **デプロイ**      | Vercel (FE) / Cloud Run (BE)     |

## 8. ストーリーボード例（利用イメージ）
1. ユーザーがFirebase Authenticationを通じてGoogleログイン
2. ダッシュボードでES作成 → Gemini添削を実行
3. 面接後に音声をアップロード → 要約とともに記録
4. 企業を登録すると、関連ESや面接が紐付いて表示される
5. カレンダーで今後のスケジュール確認
